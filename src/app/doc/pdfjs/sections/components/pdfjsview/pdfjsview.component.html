pdfjs-view

pdfjs-view allow to see a pdf page

Simple view

A simple view must have a defined pdfjsControl or pdfjsGroupControl
Control allow to control a pdf load by pdfjsControl or controlled and selected by pdfjsGroupControl
<!--In template-->
<pdfjs-view  [control]="pdfjsControl"> </pdfjs-view>

// In controller
pdfjsControl: PdfjsControl = new PdfjsControl();

ngOnInit():void {
   this.pdfjsControl.load('assets/pdfs/conditions.pdf');
}


Attributes

Fit direction and scale

Fit value, allow to define how to page is fitted when scale = 1.
Set to ViewFit.VERTICAL and the page will fitted on entire height.
Set to ViewFit.HORIZONTAL and the page will fitted on entire width.
Change scale value for define zoom in page. '1' value define no zoom and fit the page like fit value defined.
Default fit value is ViewFit.VERTICAL.
Default scale value is 1.

ViewFit.VERTICAL    ViewFit.HORIZONTAL {{scale}} times


<pdfjs-view  [fit]="'{{fit}}'" [scale]="{{scale}}"> </pdfjs-view>
Or
// In controller
ViewFit = ViewFit;
<!--In template-->
<pdfjs-view  [fit]="ViewFit.{{fit | uppercase}}" [scale]="{{scale}}"> </pdfjs-view>


Render quality

Set view render quality

1    2    3    4    5

Render pdf page use a lot of cpu load. For limit the cpu loading, use the adequate quality.


<pdfjs-view [quality]="{{quality}}"></pdfjs-view>


Text Layer

Set if text layer is enabled

true    false

<pdfjs-view [textLayer]="{{textLayer}}"></pdfjs-view>


Keys Navigation

Set if keys allow to navigate in pdf

true    false

<pdfjs-view [keysNav]="{{keysNav}}"></pdfjs-view>


MouseWheel Navigation

Set if mouseWheel allow to navigate in pdf

true    false

<pdfjs-view [mouseWheelNav]="{{mouseWheelNav}}"></pdfjs-view>
